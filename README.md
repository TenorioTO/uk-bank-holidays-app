# 🇬🇧 UK Bank Holidays

A mobile app to track and manage UK bank holidays.

Built with **TypeScript**, **React Native**, and **Expo**.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/TenorioTO/uk-bank-holidays-app.git
cd uk-bank-holidays

# Install dependencies
yarn

# Start the app
yarn start
```

This will launch the Expo development server. From there you can run the app on:

- 📱 iOS Simulator
- 🤖 Android Emulator
- 📲 Physical device via Expo Go

## Scripts

| Command        | Description                   |
| -------------- | ----------------------------- |
| `yarn start`   | Start the Expo dev server     |
| `yarn ios`     | Run on iOS simulator          |
| `yarn android` | Run on Android emulator       |
| `yarn test`    | Run tests with Jest           |
| `yarn lint`    | Lint the codebase with ESLint |

## Tech Stack

- [Expo](https://expo.dev) (v55)
- [React Native](https://reactnative.dev) (v0.83)
- [Expo Router](https://docs.expo.dev/router/introduction/) — file-based routing
- [TypeScript](https://www.typescriptlang.org)
- [Jest](https://jestjs.io) + [React Native Testing Library](https://callstack.github.io/react-native-testing-library/)
